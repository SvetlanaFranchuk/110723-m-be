Скопировать репозиторий на компьютер
1) зайти в репозиторий
2) нажать кнопку <>Code
3) клонировать ссылку в протоколе SSH (копировать)
4) перейти в GitBash в директорию, где планируем работать
5) ввести команду git clone и добавить скопированную ссылку


Как получить на локальный компьютер изменения с github.com
1) командой git pull получить изменения с гита


Какую информацию нам дают команда :
git status - показывает состояние файлов в гит репозитории
git log - показывает историю комитов в репозитории гит
